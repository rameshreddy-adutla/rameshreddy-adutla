# Hi 👋, I'm Ramesh

**Staff Software Engineer** building cloud-native microservices, APIs, and data pipelines. 14+ years across fintech, retail, and telecom — from JP Morgan and Sky to Marks & Spencer.

Day-to-day I work on backend services in Java and Spring Boot — API design, GraphQL federation, event-driven architecture, and getting things safely into production. I care about the less glamorous stuff that actually matters: observability, deployment automation, feature flags, and making sure things don't break at 2am.

In my spare time, I'm building **free AI education** for learners worldwide and contributing to open-source projects across the Apache, MCP, and GOVUK ecosystems.
